⚡ QUICK START (5 MINUTES)
================================================================================

1. INSTALL DEPENDENCIES
   cd backend
   npm install

2. CONFIGURE DATABASE
   Edit backend\.env:
   DATABASE_URL=postgresql://postgres:YOUR_PASSWORD@localhost:5432/hotel_db

3. CREATE DATABASE
   Open pgAdmin or psql:
   CREATE DATABASE hotel_db;

4. RUN MIGRATIONS
   npm run migrate

5. START SERVER
   npm run dev

6. TEST
   Visit: http://localhost:5000/health

================================================================================
🔧 CONFIGURATION REQUIRED
================================================================================

You MUST configure backend\.env with your PostgreSQL connection:

DATABASE_URL=postgresql://username:password@localhost:5432/hotel_db

Replace:
  - username: Your PostgreSQL username (usually "postgres")
  - password: Your PostgreSQL password
  - localhost: Database host
  - 5432: Database port
  - hotel_db: Database name

Example:
DATABASE_URL=postgresql://postgres:mypassword@localhost:5432/hotel_db

================================================================================
🚨 COMMON ISSUES
================================================================================

"DATABASE_URL is required"
→ Create backend\.env with your database connection string

"ECONNREFUSED"
→ Make sure PostgreSQL service is running

"Port 5000 already in use"
→ Change PORT in backend\.env to a different port

"database does not exist"
→ Create it: CREATE DATABASE hotel_db;

"password authentication failed"
→ Check your PostgreSQL password in DATABASE_URL
